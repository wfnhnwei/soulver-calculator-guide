# Soulver 3：自然语言记事本计算器，中文识别 + 永久授权 + 优惠码折上折

平时算账这件事，大家大概都经历过：要么掏出系统自带的计算器，输完一行结果就被擦掉，根本不知道刚才怎么算到这一步的；要么开个 Excel 表格，结果为了算个机票加酒店加小费，光是设公式、调列宽就花掉十分钟。Soulver 想干的事，就是把这些尴尬都省掉——它把「记事本」和「计算器」缝在了一起，你像写草稿一样随手敲下数字和描述，右侧那一列答案会跟着实时蹦出来，改一处，全篇自动重算。下面我把这套工具怎么用、目前有哪些版本、价格差异在哪、以及怎么买最划算，一次给你说清楚。

## 它到底是个什么东西

Soulver 由开发者 Zac Cohan 早在 2005 年就推出，是这类「记事本 + 自然语言计算器」形态的**原创者**。其底层由自家的端侧引擎 SoulverCore 驱动，**不依赖云端 AI**，所有运算在本地完成，对隐私和速度都更友好。整套软件体积只有 18 MB，官方强调它「不是伪装成 App 的网页」，而是用 Apple 原生技术开发，启动快、内存占用低。

它的核心交互是一个叫「答案栏」的设计——左边是你随手写的文字和数字，右边是实时算出的结果，一目了然。

举几个官方演示的例子，你大概就能体会它「读心」的感觉：

- `490 的 15%` → 自动得出 73.5
- `$3k 收入 ÷ 5 人` → $600.00
- `午餐 $55 + 25% 小费` → $68.75
- `5月5日 + 43 天` → 6月17日
- `每月房租 = $2.5k`，`话费 = $45`，`餐饮 = $50 每天 × 30 天`，最后一行写 `每月总成本` → 自动汇总为 $4,045.00

整张表是纯文本，写法自由，改一个变量，后面引用它的所有行都会跟着更新。它内置对**190 种货币**、多种计量单位、日期、时间的识别，还能通过 `= ?` 直接在行内调用 Wolfram|Alpha 查询诸如「珠穆朗玛峰高度」这类数据。

## 它和计算器、电子表格的区别

很多人会问：这玩意儿和计算器、Excel 有什么不一样？官方的回答挺直白。

**和计算器比**：计算器只能一步步算，结果不留痕，算到一半想回头改前面就抓瞎。Soulver 把整个思考过程留在页面里，你随时可以回去改一行，下游全部自动重算，**不必为了算而跳出问题本身**。

**和电子表格比**：电子表格是为「数据」设计的——一列列数值，加一堆公式把它们搅在一起。Soulver 是为「思考」设计的——你用文字和数字混着写，一行一行把事情理清楚。它是「一维的电子表格」，更接近在餐巾纸背面算账的体验，但比餐巾纸靠谱得多。

也正因为它不靠云端、不需要订阅，官方把这种哲学叫做「工具应该是你**拥有**的东西，而不是你订阅的服务」。重大付费更新平均七年才一次（2005、2010、2019、2026），中间都是免费更新。

## 主要功能一览

- **答案栏**：2005 年原创设计，文字与数字混写，右侧即时出结果，底部还有悬浮总计。
- **行引用**：双击答案即可在下方引用，改一行，引用处自动更新；以运算符开头的新行会自动接上上一行。
- **小计与总计**：双击空白行插入小计，多个小计可汇总成总计。
- **多词变量**：可给整句短语赋值，比如 `每日零食费 = $5`，再写 `每日零食费 × 365 天 每年` 自动得出 $1,825。
- **实时数据**：免费覆盖 190 种货币与 8 只热门科技股，150k+ 股票和 Wolfram|Alpha 查询需要可选的数据服务订阅。
- **行程规划**：把带日期的行标记成「时间点」，自动算出每段行程的起止日期，还能提醒你是否超出截止时间。
- **标签求和**：用 `#标签` 给行打标，可对同一标签做合计、平均等统计，适合 AA 制分账。
- **结构与样式**：支持 Markdown 标题、分隔线、高亮、四种注释样式，可隐藏不想被计算的数字，导出 PDF / HTML / CSV。

## 适合谁用，不适合谁用

**比较适合**：

- 经常要做多步推算、又不想被 Excel 公式拖住的人——比如做预算、分摊费用、估算项目成本、规划行程开销。
- 自由职业者、独立开发者、外贸从业者——需要频繁做汇率换算、时区计算、报价估算。
- 写技术文档或产品方案时，需要在文字里穿插算式并保留过程的人。
- 喜欢原生、轻量、无订阅、无云端依赖的 Mac 用户。

**需要再想想**：

- 主要在 iPhone/iPad 上做计算、且希望三端通用的用户——数码荔枝版是 Soulver 3 的 macOS 单平台版，不含 iOS。如果三端都要，建议直接走 Soulver 4 的 $59 通用购买。

- 只做最简单加减乘除、对「留痕」没有需求的人——系统计算器就够用，没必要为此付费。
